GLPI

Gestionnaire de parc informatique.

GLPI

Contexte

GLPI est un outil gratuit permettant de gérer le parc informatique d'une entreprise ou de toutes autres établissement. Lors d'un TD en première année, nous devions nous mettre dans la peau d'un technicien informatique au seins d'un lycée.


Besoin

Nous devions gérer le parc informatique, cet à dire faire l'inventaire des ordinateur dans les différentes salles des différent étages.Nous devions aussi suivre les différentes étapes d'un ticket.


Environnement

Pour que le projet soit mené à bien, nous avons créé une machine virtuelle puis installer GLPI. Cette machine virtuelle à permis de configurer GLPI et Fusion Inventory sur le parc informatique de notre machine.


Réalisation

Pour créer la machine virtuelle, nous l'avons créer sur la ferme de serveur de notre lycée. On a pu simuler le compte utilisateur, faire un ticket, et le compte technicien, où on suivait le cycle de vie d'un ticket./h6>

Bilan

Suite à ce TD nous avons pus nous familiariser avec GLPI, et nous mettre dans la peau d'un technicien, voir la différence entre OCS Inventory et Fusion Inventory, et ce TD pourrais donner à certains l'idée de faire le métier de technicien informatique.


Compétences

En réalisant cette activité, nous avons pu nous initier à la gestion du parc informatique d'une entreprise. Compétence qui nous sera utile dans le monde professionnel en nous permettant d'ouvrir notre champ de connaissance et de nous rendre plus fonctionnels.

ALLOPHONE

Site vitrine pour l'entreprise Allophone.

ALLOPHONE

Le Contexte :

L’entreprise Allophone est une entreprise basée à la Guillotière dans le 3ème arrondissement de Lyon. Spécialisée dans la téléphonie, elle offre plusieurs services tels que des impressions, photocopies, ou encore un cyber.


Le Besoin :

J’ai pu lors de ce stage créer un site vitrine pour l’entreprise en autonomie en compagnie d'un autre stagiaire dans le même domaine que moi. Ce qui leur permettra de centraliser toutes les informations les concernant : ce qu’ils font, les tarifs ou encore comment les contacter. Cela m’a aussi permis de rentrer et d’entrevoir le monde professionnel, d’apprendre de nouvelles compétences ou de renforcer mes acquis.


Environnement technologique :

Le site web que j’ai créé a été réalisé en HTML5, et il a été stylisé avec du CSS et Bootstrap.


Réalisation

J'ai réalisé ce site vitrine sur mon environnement qu'est Visual Studio Code. Pour le réaliser, nous nous sommes partager le travail, chacun faisait une page puis on rassemblait tout sur les deux pc.On a pu utiliser et se refamiliariser avec Boostrap, css et puis revoir les bases de Html.Comme c'est un site vitrine, il ne stocke aucune donnée, donc il n'y a aucune de base de donnée. Ce site est à but informatif seulement.


Bilan :

À la suite de ce travail, il reste à faire quelques modifications si nécessaire,trouver un designer pour leur créer un logo, héberger le site en ligne et y implémenter les réseaux sociaux futurs.


Compétences :
  • Travail en équipe
  • Bootstrap
  • CSS
  • HTML
